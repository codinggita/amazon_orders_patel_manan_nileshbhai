# Amazon Orders — Frontend (React Dashboard)

Frontend dashboard for the **Amazon Orders** project. **Not started yet** — begin only after backend APIs are complete and tested in Postman.

## Before you start

1. Read the full checklist: [`../README.md`](../README.md) (Part 2 — Frontend)
2. Confirm backend is running: `http://localhost:5000/api/v1/health`
3. Map every screen to a real API endpoint (no static data in final build)

## Planned stack

- Vite + React
- Tailwind CSS + MUI
- Redux Toolkit
- Axios + React Router
- Formik + Yup
- React Helmet (SEO)
